# Portfolio

Second Challenge for the BootCamp 2024 UoFT

## User's Story

```
Create a Portfolio page for the developer to show the progress and the projects made during this course.
```

## Acceptance Criteria

```
When hover on the links
Then links should be changed in style
When hover on the projects
Then there should be some shown on the projects icons
When the page is resize
Then contents should be clearly visible
WHEN click on the link to the section about their work
THEN the UI scrolls to their corresponding sections
```
